#  API de Ejemplo con FastAPI

Este proyecto implementa una **API REST** en Python usando **FastAPI** y **Pydantic**, demostrando:
- Validación de datos.
- Tipos de campos (`Optional`, `List`, `Union`).
- Conversión de tipos.
- Valores por defecto y errores de validación.



## Requisitos previos

Antes de ejecutar la API, asegúrate de tener instalado **Python 3.9 o superior** y FastAPI con Uvicorn.

Instala las dependencias con:

bash
pip install fastapi uvicorn
